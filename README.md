# 🎯 JD Skill Gap Analyzer

A web application that compares your resume against a job description 
and computes a skill match score using Python and NLP.

## Live Demo
👉 [Click here to try it](https://jd-gap-analyzer.streamlit.app)

## What It Does
- Upload your resume (PDF) or paste resume text
- Paste any Job Description
- Get instant match score, matched skills, missing skills
- Get suggestions to improve your resume

## Built With
- Python
- Streamlit
- nltk (NLP)
- PyPDF2

## How To Run Locally
pip install -r requirements.txt
streamlit run app.py
